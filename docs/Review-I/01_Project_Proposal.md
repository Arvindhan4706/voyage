## 3. SDG Mapping (UN Sustainable Development Goals)
* **SDG 8: Decent Work and Economic Growth (Target 8.9)**: Promote sustainable tourism that creates local jobs and promotes culture and local products.
* **SDG 12: Responsible Consumption and Production (Target 12.b)**: Integrate an eco-scoring engine (**Sustainability Tracker**) to quantify carbon footprints per travel itinerary, encouraging low-emission choices.

## 4. Problem Statement
Airline ticket prices fluctuate rapidly based on booking lead times, route distance, day of flight, and market demand indices. Consumers lack transparent predictive insights, while developers lack production-grade MLOps reference implementations for continuous model deployment, drift detection, and automated retraining.

## 5. Objectives
1. Build an end-to-end MLOps pipeline automating dataset generation, feature scaling, model training, and experiment tracking.
2. Develop a multi-variable price prediction regressor deployed via containerized REST API.
3. Integrate the ML serving engine into a Next.js web application for real-time inference.
4. Implement continuous model drift monitoring and GitHub Actions CI/CD automation.

## 6. Scope
* **In-Scope**: Flight price forecasting, hotel sentiment ranking, eco-sustainability scoring, FastAPI model serving, Docker containerization, MLflow tracking, DVC pipeline definitions, and GitHub Actions CI/CD.
* **Out-of-Scope**: Live payment gateway processing and booking ticket distribution.
